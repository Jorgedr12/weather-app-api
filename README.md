# Weather App API

Sitio Web: https://weather-app-api-two.vercel.app/

Aplicación sencilla para consultar el clima en tiempo real con manejo de errores y estados de red.


## APIs Gratuitas utilizadas
- [Open-Meteo Geocoding](https://open-meteo.com/en/docs/geocoding-api): Para convertir nombres de ciudades en coordenadas.
- [Open-Meteo Weather Forecast](https://open-meteo.com/en/docs): Para obtener datos meteorológicos en tiempo real.

## Tecnologías
- HTML5
- Tailwind CSS
- JavaScript (Fetch API)
